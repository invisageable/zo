# zsx

zo embeds a declarative layout system interface natively named zsx (zo Syntax Extension). The compiler handles compilation into static DOM nodes without external heavy bundle assets.

markup basics: tags, attributes, interpolation.

  ```zo
  -- type-safe markup nodes.
  fun user_account(username: str, score: int) {
    imu view: </> ::= <div class="card">
      <h2>User: {username}</h2>
      <p>Current Score: {score}</p>
    </div>;
  }
  ```

## style

Target style bindings using the lexical dollar-braces $: {} notation operators. Scoping keeps the declarations
isolated to the explicit translation package file.

  ```zo
  -- style:scoped.
  $: {
    card {
      background: #161b22;
      border-radius: 4px;
      padding: 16px;
    }
    button {
      background-color: #ff79c6;
      color: #0d1117;
      font-weight: bold;
    }
  }

  -- style:global.
  pub $: {}
  ```
