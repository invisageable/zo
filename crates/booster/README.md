# booster.

> *The `zhoo` booster.*

## about.

`zhyr` is able to translate a subset of languages to `zhoo`.

- [ ] `.js` -> `.zo` — Transforms `javascript` to `zhoo`.
- [ ] `.py` -> `.zo` — Transforms `python` to `zhoo`.
- [ ] `.wat` -> `.zo` — Transforms `wasm` to `zhoo`.
