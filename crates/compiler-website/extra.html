4 FEATURES TO HiGHLiGHT THE POWER OF zo.

#EXPRESSiVENESS.

Syntax is a key part of semantics, as Robert Virding said — « semantic is king ». Many programming languages overlook how syntax can enhance readability and convey powerful information. zo's syntax is authentic, compact and optimized for relevance and efficiency. Each symbol has a clear role, making it ideal for one-liners and code golf. See Syntax Tricks to discover the secret sauce. 


fun main() {
  imu f := fn(x) -> when x <= 1
    ? 1
    : x * f(x - 1);

  showln("{}", f(20)); -- returns `2432902008176640000`.
}

#RELiABLE.

Dealing with errors is a daily part of a programmer's journey, and many languages fail at this stage. zo is different — it's not just a tool, it's your buddy. Our advanced type inference system catches those tricky edge cases early, ensuring your code runs smoothly. And when something seems off, Zo provides clear, and rich diagnostics to help you understand what's wrong with your code. The graphical error display is designed with accessibility in mind, ensuring visibility for everyone, including those with color blindness. The zo compiler is your byakugan. you can focus on building, knowing that potential issues are handled before they even arise.

#GPUi-READY.

Building dynamic apps often requires templating, a common alternative to static development. zo includes its own templating language, inspired by E4X called zSX, offering an intuitive and universal way to create graphical user interfaces effortlessly. Take advantage of the GPU's high performance, just like html Sensei. 


fun main() {
  imu hey ::= <h1>hey!</h1>;
  imu zsx ::= <>{hey}</>;

  #dom zsx;
}

#FEATURE-SMART.

In the web world, we use the Document Object Model (DOM) API to manipulate elements on a page, but in native development, there's no DOM — We need our Shikamaru! zo provides an effective DOM API usable on both platforms — « Write less, do more » like Jquery sensei. 


fun main() {
  imu p := `#("p")
    .add("div")
    .with.class("widget");

  #dom p;
}
