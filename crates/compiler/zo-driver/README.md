# zo-driver.

> *The command-line interface of the `zo` compiler.*

## about.

The executable of the `cli` contains the following list:

- `build` — builds a `zo` project to a specific target such as `wasm` or `python`.
- `interpret` — interprets a `zo` project using a naïve interpreter.
- `repl` — reads, eval and loop, a simple ecosystem to play with the `zo` compiler.
- `help` — displays the help of the related subcommand.
