# zo-compiler.

> *The `compiler` of the `zo` programming language.*

## about.

The `compiler` runs each phase one by one to produces the right result. The compiler phase are:   

- `reading` — see the [`reader`](../zo-reader) module.
- `tokenizing` — see the [`tokenizer`](../zo-tokenizer) module.
- `parsing` — see the [`parser`](../zo-parser) module.
- `analyzing` — see the [`analyzer`](../zo-analyzer) module.
- `build` — see the [`builder`](../zo-builder) module.
- `interpreting` — see the [`interpreter`](../zo-interpreter) module.
