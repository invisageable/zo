# zhoo-compiler.

> *The `compiler` engine — it handles all the compiler phases.*

## dirmap.

```
zhoo-compiler
|-- src
    |-- phase                     # A wrapper to encapsule compiler phases.
        |-- analyzing.rs          # ...
        |-- building.rs           # ...
        |-- generating.rs         # ...
        |-- interpreting.rs       # ...
        |-- linking.rs            # ...
        |-- parsing.rs            # ...
        |-- reading.rs            # ...
        |-- tokenizing.rs         # ...
    |-- compiler.rs               # ...
    |-- lib.rs                    # ...
    |-- phase.rs                  # imports phases in the `phase` folder.
```
