## overview.

- Statically-typed language.
- Type inference.
- User-friendly report messages.
- Small binary size.
- Fast compilation time.

## development.

First, you need to clone the repo:

```
git clone https://github.com/invisageable/zo.git
```

Second, try to compile a simple program:

```
cargo run --bin zo -- build --input crates/compiler/zo-notes/samples/hello/hello.zo
```
